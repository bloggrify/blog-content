We're excited to announce the release of Bloggrify 3.0.0! This update improve existing features to enhance your blogging experience: shortcodes and built-in analytics integration.

## What's New in 3.0.0

### 1. CLI tools

Bloggrify now comes with a CLI tool to help you manage your blog. You can use it to create new posts, update existing ones, and more.

### 2. Nuxt UI integration

Bloggrify now comes with a Nuxt UI integration to help you create beautiful and modern UI for your blog.

### 3. Nuxt 4 and Nuxt-Content 3 support

Bloggrify now comes with a Nuxt 4 and Nuxt-Content 3 support. 

### 4. Author pages

Author pages are automatically generated based on the authors you define in your app.config.ts file.

And more...
* Automatic og:image generation when cover is not provided
* Use Search feature from nuxt-ui and Nuxt-Content
* Improve RSS discovery
* Improve performance using nuxt seo zeroRuntime option
